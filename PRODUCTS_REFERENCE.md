## SEO Tips for Products

1. **Titles**: Include main keyword (e.g., "Modern Grey Sectional Sofa")
2. **Descriptions**: Write 150-200 word compelling details
3. **Tags**: Add 5-10 relevant keywords
4. **Images**: Use multiple high-quality photos
5. **Collections**: Organize logically for browsing

## Pricing Strategy

- **Economy Range**: $300 - $600
- **Mid-Range**: $600 - $1,200
- **Premium Range**: $1,200 - $2,500
- **Luxury Range**: $2,500+

## Inventory Management

- Track stock levels in Shopify admin
- Set low stock alerts
- Use variant options for sizes/colors
- Enable backorder if needed

## Product Photography

- Use white or lifestyle backgrounds
- Multiple angles (front, side, top)
- Detail shots of materials
- Scale reference (person sitting)
- High resolution (minimum 1000x1000px)
- Optimize for web (under 500KB)
